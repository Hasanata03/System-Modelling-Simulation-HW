# System-Modelling-Simulation-HW
Bottle Crusher System - Modeling &amp; Simulation

This repository contains my coursework for the **KON301E System Modeling and Simulation** course. The project involves the mathematical modeling and simulation of a hydraulic bottle crusher system driven by a DC motor.

## üìÑ Project Overview
The main objective of this assignment was to design a control system for a conveyor belt mechanism that compresses plastic bottles. The project includes:
* **Mathematical Modeling:** Deriving the transfer functions for the electromechanical and hydraulic subsystems.
* **Simulink Implementation:** Block diagram representation of the system.
* **Simscape Implementation:** Physical modeling using Simscape components.
* **Simulation & Analysis:** Comparing the step responses of both models.
